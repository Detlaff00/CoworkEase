# CoworkEase

This repository contains the CoworkEase application. The React frontend resides in the `src/` directory at the project root. The backend server lives under `coworkease-backend/`.

## Frontend

Install dependencies and start the development server using Vite:

```bash
npm install
npm run dev
```

To create a production build, run:

```bash
npm run build
```

The generated files will be placed in the `dist/` folder.

## Backend

The Express backend is located in the `coworkease-backend/` folder and can be started with:

```bash
npm install
npm start
```
